Class names beginning with "gallery-" form a gallery.

.. image:: cat.jpg
   :class: gallery-1

.. image:: dog.jpg
   :class: gallery-1

.. image:: cat.jpg
   :class: gallery-2
   :scale: 50%

Some text. The next images starts a fresh gallery, despite the already
used class.

.. image:: cat.jpg
   :class: gallery-2

Figures are supported, too:

.. figure:: cat.jpg
   :class: gallery-1

.. figure:: dog.jpg
   :class: gallery-1
