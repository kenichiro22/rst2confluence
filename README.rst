=============================================
Convert reStructuredText to Confluence markup
=============================================

Supported directives
====================

- bullet list
- enumerated list
- note, warning
- image
- definition list
- simple table
- block quote
- text effect

  - strong
  - emphasis
  - monospace
- code blocks
- links


Usage
=====
::

    ./rst2confluence.py /path/to/file.rst

Installation
============
::

    sudo ./setup.py install
